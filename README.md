# CP & Algorithms Template Repository ⚡

This repository contains my personal competitive programming templates and utility functions designed to save time during coding contests and practice.

## Purpose 🎯

The goal of this repository is to provide ready-to-use templates and commonly used helper functions that reduce boilerplate and allow focusing more on problem solving rather than setup.

These templates are especially useful for:

* 🧩 LeetCode problems  
* 🍽️ CodeChef contests  
* 🏆 Codeforces contests  
* 📚 General Data Structures and Algorithms practice  

## What's Included 🧰

* Predefined type aliases (ll, vi, pii, etc.)
* Common macros for faster coding (SORT, ALL, PB, etc.)
* Frequently used functions and shortcuts
* Binary search helpers (lower_bound, upper_bound)
* Bit manipulation helpers
* Math utilities (GCD, LCM, modular constants)
* Direction arrays for grid problems
* Debug and printing utilities

## Benefits 🚀

* ⏱️ Saves time during contests  
* 🔁 Reduces repetitive code  
* ✨ Improves coding speed and readability  

## Usage 📌

You can copy the template into your solution file before starting a problem, or include specific utilities as needed.

## Notes 📝

These templates are optimized for competitive programming and interview preparation. They are continuously updated as I learn and improve.
